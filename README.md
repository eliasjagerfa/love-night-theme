# love-night-theme
This repository features a custom made vsc dark theme, based on another extension theme. It makes use of pink and purple tones, aswell as different font weights.

## How to use vsix file
1. Press Ctrl + shift + P

2. Search for *Extensions: Install from VSIX* then press Enter

3. Select the *elias-love-night-0.0.5.vsix* file from this Repo

4. Press Hold Ctrl and press K then T

5. Select *Love Night (soft values)* and **enjoy**
